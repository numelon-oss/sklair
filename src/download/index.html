Installation notes

  • The downloaded binary may need to be marked as executable on macOS/Linux: chmod +x sklair
  • macOS users may need to allow the binary in system security settings.

Package managers (coming later)

Sklair will eventually be available via package managers such as Homebrew, Winget, and APT.
For now, GitHub Releases are the canonical distribution method.

Getting started

Once Sklair is installed, sklair help tells you about available commands and what they do.

To start a project, run: 

sklair init

Sklair will guide you through a short questionnaire and generate a sklair.json file in the current directory.

This file defines how your project is built. No surprises.
